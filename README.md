# Apartment Management System

## Overview

Apartment Management System is a full-stack web application designed to simplify apartment administration and maintenance tracking. The system helps manage residents, flats, billing, payments, expenses, and complaints through a centralized dashboard.

## Features

* Resident Management
* Flat Management
* Maintenance Billing
* Payment Tracking
* Expense Management
* Complaint Management
* Dashboard Overview
* User Authentication

## Installation

### Backend

```bash
cd backend
pip install -r requirements.txt
python app.py
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
